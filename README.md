# 🎮 EduQuest – Gamified Learning Platform

EduQuest is an AI-powered gamified education platform designed to make learning interactive, engaging, and rewarding for students.

The platform transforms traditional learning into a game-like experience where students complete missions, earn XP, level up, and receive AI-powered learning guidance.

## 🚀 Features

* 🎯 **Learning Missions**

  * Lessons
  * Quizzes
  * Coding Challenges

* 🏆 **Gamification System**

  * XP points
  * Levels
  * Streaks
  * Achievements

* 📊 **Leaderboard**

  * Compare learning progress with others

* 📚 **Interactive Lessons**

  * Video based learning

* ❓ **Quiz Engine**

  * MCQ based quizzes

* 💻 **Coding Challenges**

  * Hands-on programming tasks

* 🤖 **AI Tutor Chatbot**

  * Helps students understand concepts
  * Assists with coding problems
  * Guides learning path
